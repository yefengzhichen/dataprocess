程序说明：本程序用于人类移动行为研究项目中，具体来说，是针对移动手机上网数据进行处理和可视化，方面后期分析。
代码用java编写，可视化用Swing，数据输入为手机上网数据(主要含用户id、时间、基站位置等二十多个字段)。程序整体代码量2000行左右。
(因第一次用git误删除了工程，部分文件未能恢复，待重新添加)


处理步骤：
1、数据提取清洗，对以行为单位的记录，每行二十多个字段，提取有用字段。
2、所有文件汇总整理。现有的数据是每天一个文件，按时间格式保存的35个文件，这一步的工作是将所有记录汇总整理成一个文件，以方便后期处理。
3、提取每个用户的所有记录，记录其位置变化，即所使用的基站位置变化。
4、对用户的移动轨迹进行展示，使用的是java的swing。
